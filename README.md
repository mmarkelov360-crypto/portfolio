# Персональное портфолио разработчика

Современное интерактивное портфолио на Next.js с 3D-элементами и адаптивным дизайном.

## 🚀 Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **Three.js / React Three Fiber** - 3D графика
- **EmailJS** - отправка сообщений
- **Next SEO** - SEO оптимизация

## ✨ Особенности

- ✅ Адаптивный дизайн
- ✅ Интерактивная 3D сцена
- ✅ Плавные анимации
- ✅ Форма обратной связи
- ✅ SEO оптимизация
- ✅ Оптимизация производительности
- ✅ Кроссбраузерность
- ✅ Соответствие WCAG
